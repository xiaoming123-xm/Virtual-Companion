# 场景配置系统

> 基于对 `types/vrm.ts`、`useVRMStore.ts`、`AIStage.tsx`、`Character.tsx`、`BackgroundSystem.tsx`、`VRMRenderSettings.tsx` 的完整代码审查

---

## 一、现状梳理

### 现有能力（VRMRenderConfig）

- ✅ 光照三件套（主光源/环境补光/边缘补光）
- ✅ 后处理特效（Bloom/DOF/Vignette/ContactShadows）
- ✅ 背景图片（硬编码 `/BG/` 路径，select 枚举）
- ✅ 行为控制（自动眨眼/lookAt 模式）

### 缺失能力

| 缺失能力 | 当前状态 | 影响 |
|---|---|---|
| 角色位置偏移 | Character.tsx 硬编码自动对齐逻辑 | 无法通过设置/AI命令调整角色站位 |
| 角色缩放 | 无 | 不同 VRM 模型大小不一致，无法统一 |
| 相机目标/距离/FOV | GenshinControls 硬编码参数 | 无法按角色或场景调整默认视角 |
| 天空盒 | 仅支持背景贴图（2D），无 HDRI/CubeMap | 缺少立体环境光照 |
| 地面 | `enableContactShadows` 仅是投影 | 场景搭建能力弱 |
| 背景扩展 | 硬编码 3 张枚举图 | 用户和 AI 均无法指定任意背景 |
| 角色旋转偏移 | 硬编码 `rotation.y = Math.PI` | 无法微调角色朝向 |

## 二、新增类型定义

### 2.1 VRMSceneConfig（场景配置）

```typescript
// types/vrm-scene.ts — 新文件

/**
 * 场景背景类型
 */
export type BackgroundType =
  | 'none'          // 纯黑/透明
  | 'image'         // 2D 贴图（现有）
  | 'hdri';         // HDRI 环境贴图（球形）

/**
 * 场景背景配置
 */
export interface BackgroundConfig {
  type: BackgroundType;
  url?: string;              // 文件路径或 URL（image/hdri）
  intensity?: number;        // HDRI 环境光强度（仅 hdri）
}

/**
 * 地面配置
 */
export interface GroundConfig {
  enabled: boolean;
  type: 'none' | 'contact-shadow' | 'plane' | 'grid';
  color?: string;
  opacity?: number;
  size?: number;
  shadowOpacity?: number;    // contact-shadow
  shadowBlur?: number;       // contact-shadow
}

/**
 * VRM 场景配置（新增，持久化）
 */
export interface VRMSceneConfig {
  background: BackgroundConfig;
  ground: GroundConfig;
}

export const DEFAULT_VRM_SCENE_CONFIG: VRMSceneConfig = {
  background: {
    type: 'image',
    url: 'BG_AronaRoom.jpg',
  },
  ground: {
    enabled: true,
    type: 'contact-shadow',
    shadowOpacity: 0.4,
    shadowBlur: 2.5,
  },
};
```

### 2.2 VRMCharacterConfig（角色配置）

```typescript
/**
 * VRM 角色变换配置
 * 应用于 Character 组件的根节点，在自动对齐之后叠加
 */
export interface VRMCharacterConfig {
  positionOffset: [number, number, number]; // 位置偏移 [x, y, z]
  scale: number;                             // 缩放（1.0 = 原始大小）
  rotationY: number;                         // 旋转偏移（弧度，叠加在 Math.PI 之上）
}

export const DEFAULT_VRM_CHARACTER_CONFIG: VRMCharacterConfig = {
  positionOffset: [0, 0, 0],
  scale: 1.0,
  rotationY: 0,
};
```

**Character.tsx 集成方式：**

```typescript
// 在自动对齐后，追加叠加变换
useEffect(() => {
  if (!vrm || !hasAdjustedPosition.current) return;
  const { positionOffset, scale, rotationY } = characterConfig;
  vrm.scene.position.x += positionOffset[0];
  vrm.scene.position.y += positionOffset[1];
  vrm.scene.position.z += positionOffset[2];
  vrm.scene.scale.setScalar(scale);
  vrm.scene.rotation.y = Math.PI + rotationY;
}, [vrm, characterConfig, hasAdjustedPosition.current]);
```

### 2.3 VRMCameraConfig（相机配置）

```typescript
/**
 * VRM 相机默认配置
 * 控制 GenshinControls 的初始参数
 */
export interface VRMCameraConfig {
  target: [number, number, number];  // 注视目标（通常腰部~胸部高度）
  defaultDistance: number;            // 默认距离
  minDistance: number;                // 最小距离
  maxDistance: number;                // 最大距离
  fov: number;                        // 视野角（度）
  initialAzimuth: number;            // 初始水平角度（偏航，弧度）
  initialPolar: number;              // 初始垂直角度（俯仰，弧度）
}

export const DEFAULT_VRM_CAMERA_CONFIG: VRMCameraConfig = {
  target: [0, 0.9, 0],
  defaultDistance: 2.0,
  minDistance: 0.8,
  maxDistance: 5.0,
  fov: 45,
  initialAzimuth: 0,
  initialPolar: Math.PI / 3,
};
```

## 三、Store 扩展方案

**推荐方案：扩展到现有 `useVRMStore`**

```typescript
interface VRMStore {
  // 现有
  renderConfig: VRMRenderConfig;         // 光照/后处理/眨眼/lookAt
  setRenderConfig(updates): void;

  // 新增
  sceneConfig: VRMSceneConfig;           // 背景/地面
  setSceneConfig(updates): void;

  characterConfig: VRMCharacterConfig;   // 位置/缩放/旋转
  setCharacterConfig(updates): void;

  cameraConfig: VRMCameraConfig;         // 相机参数
  setCameraConfig(updates): void;

  // 运行时（不持久化）
  runtime: VRMRuntimeState;
}

// partialize 持久化所有配置，但不持久化 runtime
partialize: (state) => ({
  renderConfig: state.renderConfig,
  sceneConfig: state.sceneConfig,
  characterConfig: state.characterConfig,
  cameraConfig: state.cameraConfig,
})
```

## 四、UI 设置面板扩展

将 `VRMRenderSettings.tsx` 拆分为多个 Tab：

```
[光照] [场景] [角色] [相机] [高级]
```

| Tab | 控件 |
|---|---|
| **光照** | 主光源/环境补光/边缘补光（已有） |
| **场景** | 背景类型选择、URL输入、地面样式 |
| **角色** | X/Y/Z 位置滑条、缩放滑条、旋转滑条 |
| **相机** | 目标高度、默认距离、FOV、最小/最大距离 |
| **高级** | 后处理/阴影/眨眼/lookAt（原高级折叠区）|

## 五、Director 可控性（AI 命令扩展）

这些配置不仅支持用户手动调整，还支持 **AI 通过 `perform_actions` 工具调用**：

```typescript
// VRMCommand 扩展（补充到 commands.ts）

// 场景命令
| { type: 'SET_BACKGROUND'; payload: BackgroundConfig }
| { type: 'SET_GROUND'; payload: Partial<GroundConfig> }

// 角色变换命令
| { type: 'SET_CHARACTER_POSITION'; payload: { offset: [number, number, number] } }
| { type: 'SET_CHARACTER_SCALE'; payload: { scale: number } }

// 相机命令（Director 通道，立即执行，不排队）
| { type: 'CAMERA_MOVE'; payload: { preset?: CameraPreset; target?: [number,number,number]; distance?: number } }
| { type: 'CAMERA_FOV'; payload: { fov: number; duration?: number } }
```
