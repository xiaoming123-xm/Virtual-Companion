## 里程碑总结

```
阶段0 (独立) ──→ 场景配置 UI 可用
       ↓
阶段1 ──→ 基础对话跑通（useStream + text）
       ↓
阶段2 ──→ VRM 表演跑通（Director + 状态机）
       ↓
阶段3 ──→ 完整链路打通（TTS + 双向同步）
       ↓
阶段4+ ──→ 增强功能逐步添加
```

> **建议执行顺序：阶段0 和 阶段1 可以并行进行（前者纯前端，后者需要前后端配合）。阶段2 依赖阶段1 的 useStream 基础。阶段3 依赖阶段2 的 Director 基础。**
